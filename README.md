# 🚀 AI Study Agent: Intelligent Knowledge Capture System

A powerful AI-driven tool developed to assist students and developers in capturing, analyzing, and logging technical questions in real-time. This project was built as a personal initiative during my ICT studies to bridge the gap between instant answers and long-term knowledge retention.

## 🌟 Key Features
- **Smart Snipping:** Select any region on your screen to extract technical content using Tesseract OCR.
- **AI Reasoning:** Powered by **Groq (Llama-3.1-8b)** to provide not just answers, but logical explanations.
- **Autonomous Logging:** Automatically saves every interaction into a persistent study log.
- **Analytical Dashboard:** A modern UI built with **Streamlit** to visualize study progress and review complex concepts.
- **Controlled Demo Environment:** Includes a custom HTML/JS quiz page for safe and effective testing.

## 🛠 Tech Stack
- **Language:** Python 3.12
- **GUI:** Tkinter (Custom transparent overlay)
- **OCR:** PyTesseract (Google Tesseract Engine)
- **AI Engine:** Groq API (Llama Models)
- **Web Dashboard:** Streamlit & Pandas
- **Automation:** Selenium & Webdriver-Manager
